The "git init" command creates a .git directory in the present folder.
This need to be done in order a astart using git to track the changes of your project

The git Staging area is a virtual area where you add files that you want their changes
to be committed, while the working directory is a directory that contains, the files in the staging area
and other associated files.

Its advisable to commit changes once a file is added to the Staging Area and a
logical change has been made

Git Branch comes handy when you want to try out a new feature than youre not sure of
or when you want to create a different language version.

Drawing Git Diagrams helps me vissualize the whole structure of my git repository. I can view
commits and their relationships, and also branches and their connectivity.
